# FGO-script

script for lazy people (ex: me)

## Requirements

```
pip install mss
pip install pyautogui
pip install opencv-python
```

## Usage
```
將模擬器放至最大(非全螢幕)，畫面停留在入關前，執行 main.py，然後就可以看到爛爛的結果0u0
(若有雙螢幕縮放需調整成100%)
ult指令只能放每個wave的最後
```
